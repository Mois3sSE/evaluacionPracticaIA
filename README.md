# Proyecto IA - Visualizador de Algoritmos de Búsqueda

Este proyecto implementa y compara algoritmos de búsqueda (No informada, Informada, Local y Adversaria) aplicados a cuatro entornos: FrozenLake, Sokoban, 8 Reinas y Gato.

El proyecto está construido bajo una arquitectura **MVC (Modelo-Vista-Controlador)** estricta para separar la interfaz gráfica de la lógica matemática.

---

## Flujo de Trabajo (Git & Ramas)

La rama `main` está protegida. Nadie puede hacer push directo a esta rama para evitar romper el código. El flujo de trabajo será el siguiente:

1. Baja los últimos cambios: `git checkout main` y luego `git pull`
2. Pásate a tu rama asignada: `git checkout feature/tu-rama`
3. Trabaja y haz tus commits: `git commit -m "Lo que hiciste"`
4. Sube tu rama: `git push origin feature/tu-rama`
5. Abre un **Pull Request (PR)** en GitHub para integrar tu código al Controlador.
